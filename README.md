# Etbanx

> Un espacio digital de conocimiento, experimentos y proyectos — construido con Astro y Tailwind CSS.

## 🧭 ¿Qué es Etbanx?

Etbanx es un sitio personal que funciona como base de conocimiento ("digital garden"), donde se documentan ideas, investigaciones, laboratorios de experimentación y proyectos en curso. El objetivo es tener un único lugar organizado para pensar, escribir y construir en público.

## 🗂️ Secciones principales

- **Knowledge** — Notas y artículos de conocimiento general, conceptos y referencias.
- **Labs** — Experimentos técnicos, pruebas de concepto y sandboxes.
- **Research** — Investigaciones profundas sobre temas específicos.
- **Projects** — Proyectos terminados o en desarrollo.
- **Notes** — Apuntes rápidos, ideas sueltas y borradores.

## 🛠️ Stack tecnológico

- [Astro](https://astro.build/) — Framework principal del sitio.
- [Tailwind CSS](https://tailwindcss.com/) — Estilos y diseño.
- TypeScript — Tipado para contenido y componentes.
- Markdown / MDX — Formato de contenido.

## 🚀 Cómo correr el proyecto localmente

Clonar el repositorio:

```bash
git clone https://github.com/[tu-usuario]/etbanx.git
cd etbanx
```

Instalar dependencias:

```bash
npm install
```

Levantar servidor de desarrollo:

```bash
npm run dev
```

El sitio quedará disponible en `http://localhost:4321`.

## 📦 Comandos disponibles

| Comando           | Acción                                           |
|--------------------|---------------------------------------------------|
| `npm install`      | Instala las dependencias                          |
| `npm run dev`      | Inicia el servidor de desarrollo                  |
| `npm run build`    | Genera la build de producción en `./dist/`        |
| `npm run preview`  | Previsualiza la build de producción localmente    |
